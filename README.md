## 🚀 How to Deploy on GitHub Pages (Free Hosting)

1. **Create a new GitHub repository**
   - Go to [github.com](https://github.com) → New Repository
   - Name it: `vaishnavi-portfolio` (or `<your-username>.github.io` for your main site)
   - Set it to **Public**

2. **Upload your files**
   - Click "Add file" → "Upload files"
   - Drag and drop `index.html` and `README.md`
   - Click "Commit changes"

3. **Enable GitHub Pages**
   - Go to repository **Settings** → **Pages**
   - Under "Source", select **Deploy from a branch**
   - Choose `main` branch → `/ (root)` folder
   - Click **Save**

4. **Your site is live!**
   - URL will be: `https://<your-username>.github.io/vaishnavi-portfolio/`
   - (Takes ~2 minutes to go live)
